# Awwwards技術完全習得ロードマップ

## 🎯 最終目標
[Layrid](https://layrid.tomoyaokada.com/) のような、Awwwardsに掲載されるレベルのクリエイティブWebサイトを自力で構築できるようになる。

---

## Phase 1: Foundation (基礎) - 2-3週間

### Lesson 1.1: HTML/CSS基礎 ✅
- **目標**: Webページの構造と装飾の基本を理解する
- **技術**: HTML5, CSS3
- **成果物**: シンプルな自己紹介ページ
- **Challenge**: 色、サイズ、配置を自由に変更できる

### Lesson 1.2: Box Model & Flexbox
- **目標**: レイアウトの仕組みを理解する
- **技術**: CSS Box Model, Flexbox
- **成果物**: カード型レイアウトのギャラリーページ
- **Challenge**: 3カラムレイアウトを作る

### Lesson 1.3: CSS Grid & Positioning
- **目標**: 複雑なレイアウトを自在に組む
- **技術**: CSS Grid, position (absolute/relative)
- **成果物**: 雑誌風レイアウトのページ
- **Challenge**: 要素を重ねて配置する

### Lesson 1.4: Transition & Transform
- **目標**: 動きのある要素を作る
- **技術**: CSS transition, transform
- **成果物**: ホバーで拡大するカードUI
- **Challenge**: 回転・スケール・移動を組み合わせる

### Lesson 1.5: レスポンシブデザイン
- **目標**: スマホ・タブレット・PCで最適表示
- **技術**: Media Queries, Viewport
- **成果物**: どのデバイスでも美しいページ
- **Challenge**: ブレイクポイントを3つ設定する

---

## Phase 2: JavaScript & Interactivity (動的表現) - 3-4週間

### Lesson 2.1: JavaScript基礎
- **目標**: プログラミングの基本を理解する
- **技術**: 変数, 関数, 条件分岐, ループ
- **成果物**: 簡単な計算機
- **Challenge**: ボタンを押したら数字が増えるカウンター

### Lesson 2.2: DOM操作
- **目標**: JavaScriptでHTMLを操作する
- **技術**: querySelector, addEventListener, classList
- **成果物**: クリックで色が変わるボタン
- **Challenge**: モーダルウィンドウを作る

### Lesson 2.3: スクロールイベント
- **目標**: スクロールに反応する要素を作る
- **技術**: scroll event, IntersectionObserver
- **成果物**: スクロールでフェードインする要素
- **Challenge**: スクロール量に応じて背景色を変える

### Lesson 2.4: Fetch API & 非同期処理 ✅
- **目標**: 外部データを取得して表示する
- **技術**: fetch, async/await, JSON
- **成果物**: APIから画像を取得して表示するギャラリー
- **Challenge**: ローディング表示を実装する

---

## Phase 3: Modern Stack (モダンツール) - 3-4週間

### Lesson 3.1: Next.js入門 ✅
- **目標**: モダンなフレームワークを使う
- **技術**: Next.js, React基礎
- **成果物**: ページ遷移のあるサイト
- **Challenge**: 動的ルーティングを実装する

### Lesson 3.2: Tailwind CSS
- **目標**: 高速でスタイリングする
- **技術**: Tailwind CSS, Utility-first CSS
- **成果物**: Tailwindで作るランディングページ
- **Challenge**: カスタムカラーを設定する

### Lesson 3.3: Framer Motion
- **目標**: 宣言的にアニメーションを作る
- **技術**: Framer Motion
- **成果物**: ページ遷移アニメーション
- **Challenge**: スクロールトリガーアニメーション

## Phase 4: Creative Coding (クリエイティブ表現) - 4-6週間

### Lesson 4.1: GSAP基礎
- **目標**: プロレベルのアニメーションライブラリを使う
- **技術**: GSAP, ScrollTrigger
- **成果物**: パララックススクロール
- **Challenge**: ピン留めセクションを作る

### Lesson 4.2: Three.js入門
- **目標**: 3D表現をWebに組み込む
- **技術**: Three.js, WebGL
- **成果物**: 回転する3Dオブジェクト
- **Challenge**: マウスに追従する3D球体

### Lesson 4.3: React Three Fiber
- **目標**: ReactでThree.jsを扱う
- **技術**: React Three Fiber, Drei
- **成果物**: インタラクティブな3Dシーン
- **Challenge**: クリックで色が変わる3Dモデル

### Lesson 4.4: Shader入門
- **目標**: カスタムビジュアルエフェクトを作る
- **技術**: GLSL, ShaderMaterial
- **成果物**: 波打つグラデーション背景
- **Challenge**: マウス位置でエフェクトを変える

### Lesson 4.5: Lenis & Smooth Scroll
- **目標**: 滑らかなスクロール体験を実装する
- **技術**: Lenis, Custom Scrollbar
- **成果物**: ヌルヌル動くスクロール
- **Challenge**: スクロール速度をカスタマイズ

## Phase 5: Polish & Performance (仕上げ) - 2-3週間

### Lesson 5.1: パフォーマンス最適化
- **目標**: 60fps を維持する
- **技術**: Lighthouse, React.memo, useMemo
- **成果物**: 最適化されたサイト
- **Challenge**: Core Web Vitals を全て緑にする

### Lesson 5.2: 画像・動画最適化
- **目標**: 重いメディアを軽量化する
- **技術**: Next.js Image, WebP, Lazy Loading
- **成果物**: 高速ロードするギャラリー
- **Challenge**: 動画を背景に使う

### Lesson 5.3: SEO & メタタグ
- **目標**: 検索エンジンに最適化する
- **技術**: Next.js Metadata, OGP
- **成果物**: SNSシェア対応サイト
- **Challenge**: 動的OG画像生成

### Lesson 5.4: デプロイ
- **目標**: 世界に公開する
- **技術**: Vercel, カスタムドメイン
- **成果物**: 本番環境のURL
- **Challenge**: 独自ドメインを設定する
