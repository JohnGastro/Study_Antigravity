# Lesson 2.4: Fetch API & 非同期処理

## 🎯 目標
外部のデータ（API）を取得して、画面に表示する。

## 📝 学んだこと

### 1. Fetch API (`fetch`)
- **役割**: インターネット上のデータを取りに行く「使いっ走り」。
- **構文**: `fetch('URL')`

### 2. 非同期処理 (`async` / `await`)
- **`async function`**: 「この関数は時間がかかる処理を含みます」という宣言。
- **`await`**: 「データが届くまで待っててね」という合図。
- **なぜ必要？**: これがないと、データが届く前に次の行を実行してしまい、エラーになるから。

### 3. JSONデータの扱い
- **`response.json()`**: 届いたデータ（封筒）を開けて、中身を取り出す作業。
- これも時間がかかるので `await` が必要。

### 4. UX（ユーザー体験）
- **ローディング表示**: 待ち時間に「読み込み中...」などを表示して、ユーザーを安心させる。

## 💻 成果物
- **Dog Fetcher**: ボタンを押すとランダムな犬の画像が表示されるアプリ。
