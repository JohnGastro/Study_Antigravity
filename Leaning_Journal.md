- Lesson 2.1: JavaScript基礎
- ボタンを押したら数字が増えるカウンターを作成

### ✅ 達成したこと

#### 技術面
1. **JavaScriptの役割**
   - HTML（骨組み）、CSS（服）、JavaScript（操り人形師）
   - JS自体に描画機能はなく、HTML/CSSを操作して画面を変える

2. **変数（箱）**
   - `const`: 中身を変えられない箱（定数）
   - `let`: 中身を変えられる箱（変数）
   - なぜ使い分ける？ → 「変えないぞ」という意思表示と安全性

3. **要素の取得**
   - `document.getElementById('名前')`: 名札（id）で要素を探す
   - `const countText = ...`: 探した要素（黒板）を箱に入れておく

4. **イベントリスナー**
   - `addEventListener('click', ...)`: クリックを監視する
   - 「ボタン君、クリックされたら、この中のことをやってね」という手紙

5. **DOM操作**
   - `innerHTML`: 要素の中身（文字）を書き換える
   - `countText.innerHTML = count`: 計算した数字を表示する

#### 成果物
- ✅ `counter.html` - ボタンで数字が増えるカウンター

- Lesson 2.2: DOM操作
- クリックで色が変わるボタンを作成

### ✅ 達成したこと

#### 技術面
1. **CSSセレクタの復習**
   - `.class` (ドット) vs `#id` (シャープ)
   - 優先順位（詳細度）: ID > Class
   - `#box.active` で優先順位を上げるテクニック

2. **DOM操作（スタイル変更）**
   - `element.style.backgroundColor`: 直接スタイルを書き換える
   - プロパティ名はキャメルケース（`background-color` → `backgroundColor`）

3. **DOM操作（クラス操作）**
   - `element.classList.add('class')`: クラスを追加
   - `element.classList.toggle('class')`: クラスをつけ外し（スイッチ）
   - プロの現場では `style` 直接変更より `classList` が好まれる

#### 成果物
- ✅ `color_changer.html` - クリックで色と形が変わるボタン
